# Nuxt-AI-Chat
Nuxt UI Chat


## Goal

- Nuxt Based AI Chat App with API Backend
- Similar to https://github.com/mckaywrigley/chatbot-ui

## Tasks

## Setup

Make sure to install the dependencies:

```bash

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```
